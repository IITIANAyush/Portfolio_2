Duration: Jun ’24 – Aug ’24

Designed a hybrid vehicle that switches between car and drone modes with reorienting motor arms. Implemented ATmega-based power system, coaxial drives, and I2C communication.

  • Designed an innovative vehicle that can function as both car and drone by changing orientation of the motorized arms.
  • Optimized for Seamless movement through any terrain by transforming modes and controls by single radio transmitter.
  • Designed and assembled the Model with precise dimensions using Fusion 360, ensuring seamless hardware transition.
  • Designed the circuit model and iterated code using online simulator like TinkerCAD to avoid mechanical constrains.
  • Designed the circuit board with good power distribution using ATmega microcontroller and other electronic components.
  • Used I2C communication protocol between microcontrollers to distribute computation load and boost efficiency.
  • Seamlessly integrated Servos, Brushless DC motors, power distribution board andMPU6050 with microcontrollers.
  • Created a seamless coaxial drive for both propeller and tyre from di↵erent driving source using multiple ball bearings.
